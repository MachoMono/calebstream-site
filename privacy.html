Privacy Policy

Last updated: June 19, 2026

This Privacy Policy describes how the CalebStillStreams Pipeline application ("the App") handles data. The App is a personal automation tool operated solely by Caleb (calebstillstreams) for managing his own YouTube content.

What the App Does

The App uses the YouTube Data API to automate post-stream updates to videos on the CalebStillStreams YouTube channel. Specifically, it:

  • Reads recently published YouTube videos on the owner's channel
  • Uploads custom thumbnails to those videos
  • Updates video titles, descriptions, and tags

Data Accessed

The App accesses YouTube account data belonging only to the channel owner (Caleb). No data is collected from viewers, subscribers, or any third parties.

Data Storage

OAuth credentials required for YouTube API access are stored securely in Google Cloud Secret Manager within the owner's own Google Cloud project. No personal data is stored in external databases or transmitted to third parties.

Data Sharing

No data is sold, rented, or shared with any third parties. The App is a private tool used exclusively by its owner.

Third-Party Services

The App interacts with the following external APIs:

  • YouTube Data API v3 — to update video metadata and thumbnails
  • Twitch EventSub — to receive stream-end notifications

Use of these APIs is governed by their respective terms and privacy policies.
